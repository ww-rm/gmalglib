# gmalglib 用户指南

欢迎来到 gmalglib! 一个国密算法的 Python 扩展库, 提供一些原始基础算法, 开箱即用.

## 安装

```bat
pip install gmalglib
```

## 包含的算法

- SM3 密码杂凑算法
- SM4 分组密码算法
- ZUC 序列密码算法
- SM2 椭圆曲线公钥密码算法
    - 签名验签
    - 加密解密
